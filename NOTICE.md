# Notice

Peanut Illustrations is an AI-agent skill by Aishwarya Ashok.

It is an independent reimagining, **inspired by** [ian-xiaohei-illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations) by Ian (MIT-licensed). The original is built around the "小黑 (Xiaohei)" character for Chinese article illustrations; this project replaces that character and visual language with its own (a deadpan peanut, a single warm-orange accent, English output, and a short-prompt input model).

No artwork from the original repository is redistributed here.

When redistributing or adapting this repository, please keep the `Peanut Illustrations` name or provide attribution, and keep the inspiration credit to the original `ian-xiaohei-illustrations` project.
