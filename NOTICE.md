# Notice

Peanut Illustrations is an AI-agent skill by Aishwarya Ashok.

It is **adapted and translated from** [ian-xiaohei-illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations) by Ian (MIT-licensed). The original is built around the "小黑 (Xiaohei)" character for Chinese article illustrations.

Portions of this project are adapted from the original; the rest is new. This project changes the character (a deadpan peanut in place of Xiaohei), reduces the color system to a single warm-orange accent, switches the output to English, adds a short-prompt single-image input model, and adds a local watermark script. **No artwork from the original repository is redistributed here** — all images are independently generated.
