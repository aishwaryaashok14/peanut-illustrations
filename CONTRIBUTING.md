# Contributing

Thanks for wanting to improve Peanut Illustrations.

Good contributions are usually small and visual:

- stronger example prompts
- cleaner reference guidance
- better QA checks for generated images
- gallery examples that match the style
- install notes for Claude Code, Codex, or other agent tools

## Style guardrails

Keep the visual language consistent:

- one idea per image
- white background
- black hand-drawn line art
- one warm-orange accent
- lots of whitespace
- short handwritten labels
- the peanut performs the core action

Avoid adding generic infographic language, cute sticker-style mascot behavior, dense diagrams, or extra color systems.

## Pull requests

Before opening a PR:

1. Keep the change focused.
2. Check that examples still point to real files.
3. If you add images, make sure they include the `© peanut-illustrations-by-aish` watermark.
4. Mention whether you tested the skill in Claude Code, Codex, or both.

If you are not sure whether an idea fits, open an issue first.
