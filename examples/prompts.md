# Prompt Examples

Copy any of these straight into your agent. Each one is a short idea — the skill invents the metaphor, puts the peanut to work, and draws one image.

## Draw one idea

```text
Use $peanut-illustrations to draw a concept illustration for this idea:

Too many tools, not enough focus.

Keep it strange but clean, the peanut must perform the core action,
and use at most a few short labels.
```

## A judgment / opinion

```text
Use $peanut-illustrations to draw one 16:9 concept illustration for:

Trust isn't announced — it's laid down one piece of evidence at a time.

Deadpan but clean. The peanut carries the core action. English labels, short.
```

## A process / workflow

```text
Use $peanut-illustrations to draw an image for:

Turn one raw idea into three kinds of content.

Don't draw a formal flowchart. Invent a fresh low-tech metaphor and let the
peanut do the work. One warm-orange accent only.
```

## A before / after

```text
Use $peanut-illustrations to draw a before/after concept image for:

From scattered notes to one clear decision.

Chaos on the left, calm on the right, one orange arrow between. The peanut
makes the change happen.
```

## A feeling / state

```text
Use $peanut-illustrations to draw a concept illustration for:

The quiet panic of an inbox that never empties.

Make the peanut the one stuck in it. Keep it sparse and a little absurd.
```

## Several ideas at once

```text
Use $peanut-illustrations to draw one image each for these ideas:
- information overload
- shipping beats polishing
- a small loop you can repeat
- building trust slowly

One composed image per idea — not a collage. Generate them separately.
```
